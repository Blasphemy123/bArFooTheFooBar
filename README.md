### 3.存在的bug

由于本人技术力有限，有已知的bug但是无法修复，故在此提前告知：死亡后死亡画面可能会延长导致文本提示无法正常显示，重新拉伸边框大小即可修复

死亡会保留装备，退出游戏成就依然保留（并可以通过主界面上按钮进入成就画面查看）

由于是java语言编写，需要在有jdk的环境下运行
